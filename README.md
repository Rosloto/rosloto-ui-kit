# Rosloto UI Kit

Reusable UI components, design tokens, and frontend standards used across Rosloto products.

[![CI](https://github.com/Rosloto/rosloto-ui-kit/actions/workflows/ci.yml/badge.svg)](../../actions)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Overview

Rosloto UI Kit provides a consistent design system for building lottery, gaming, and customer-facing applications.

The library includes:

- Reusable React components
- Design tokens
- Accessibility guidelines
- Storybook documentation
- TypeScript support
- Mobile-first design patterns

## Components

| Component | Description |
|------------|-------------|
| Button | Primary actions and navigation |
| Input | Text fields and form controls |
| Modal | Dialogs and confirmations |
